# Security Policy

## Scope
ATIRF is an early-stage open cyber defense project. Please report security issues responsibly and avoid public disclosure until there is time to evaluate the impact.

## How to Report
For now, open a private security report through GitHub Security Advisories if enabled, or contact the maintainer privately before filing a public issue.

## Guidance
- do not include secrets in reports
- provide reproduction steps when possible
- identify whether the issue affects demo-only flows or production-oriented paths
- note any impact on telemetry, CTI, auth, data exposure, or response actions
